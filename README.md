## About The Project

Travel planning can be overwhelming ‚Äî from comparing endless flight and hotel options to figuring out the best local attractions.  
This **AI-Enhanced Travel Planner** aims to solve this by combining **Artificial Intelligence, Machine Learning, and API integrations** to create a seamless, personalized travel experience.

**Tailored2Trips** is an intelligent web application designed to make trip planning effortless.  

### üîπ Core Idea  
Users can simply describe their travel goals in natural language (e.g., *‚ÄúPlan a 7-day beach vacation in Spain with a mix of hiking, nightlife, and local food experiences‚Äù*), and the system automatically generates a **custom itinerary** tailored to their preferences.

‚ú® **Key Highlights**:  
- AI-driven personalized itineraries.
- Natural Language Processing (NLP) for understanding user preferences.
- API integrations with flights, hotels, attractions, and weather data.  
- Interactive drag-and-drop itinerary builder with map visualization.
- Smart optimization for cost, time, and travel experience.


### Built With 

- **Frontend**:  
  - React.js
  - Google Maps API  

- **Backend**:  
  - Node.js
  - Python (Flask/FastAPI for ML services)  

- **AI/ML**:  
  - spaCy / Hugging Face Transformers (NLP)  
  - XGBoost / Random Forests (itinerary optimization)  
  - Clustering (KMeans for grouping attractions)  

- **Database**:  
  - MongoDB / PostgreSQL  

- **APIs Integrated**:  
  - Google Places  
  - Skyscanner  
  - Airbnb / Booking  
  - OpenWeather
